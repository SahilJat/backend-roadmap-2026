## Setup — Do This First

You don't need to install anything yet. We'll use a browser-based editor.

1. Go to **[replit.com](https://replit.com)**
2. Click **"Create Repl"**
3. Choose **Node.js** as the template
4. Name it anything you want
5. You'll see a file called `index.js` — write all your code there
6. Hit the **Run** button to execute it
7. Output appears in the console on the right

---

## The Challenge

Write a JavaScript program that does all of the following:

**Task 1.**
Create a `const` variable called `serverName` and assign it the value `"MyFirstServer"`. Log it to the console.

**Task 2.**
Create a `let` variable called `activeUsers` and assign it the value `0`. Then reassign it to `42`. Log the final value.

**Task 3.**
Try to reassign your `serverName` const to something else. Write the line, then **comment it out** with `//` so it doesn't crash your program. Write a comment next to it explaining WHY it would fail.

**Task 4.**
Create a `const` called `isServerRunning` and set it to `true`. Log it.

**Task 5.**
Write a block of code using `{ }` curly braces. Inside the block, declare a `let` variable called `blockMessage` with any string value. Log it **inside** the block. Then try to log it **outside** the block and write a comment explaining what would happen and why.

---

## What You Are Looking For
- Correct use of `const` vs `let`
- You understand why reassigning `const` fails
- You understand block scope
- Clean, readable code with good variable names
