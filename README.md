# 🚀 Zero to Job-Ready: The Ultimate Backend Developer Roadmap

Welcome! If you are a complete beginner looking to break into backend development, you are in the right place. 

I created this repository to provide a structured, step-by-step curriculum that takes you from having zero coding experience to becoming a job-ready backend developer. We will start with **JavaScript** to build a solid foundation, and eventually level up to **TypeScript** — exactly how modern tech companies build their applications.

---

## 📖 How to Use This Course

This repository is designed to be followed sequentially. **Do not skip ahead.**

1.  **Check the Curriculum:** Scroll down to see the roadmap.
2.  **Navigate to the Lesson:** Click into the corresponding folder for the phase and topic you are on.
3.  **Read the Lesson Notes:** Each folder contains its own `README.md` with the theory, explanations, and a "Why Before How" breakdown.
4.  **Complete the Challenge:** Most lessons include a small coding challenge. Try to solve it yourself before looking at the provided solution files!

---

## 🛠️ Tech Stack Covered

By the end of this curriculum, you will be confident building with:
*   **Language:** JavaScript (ES6+) -> TypeScript
*   **Runtime:** Node.js
*   **Framework:** Express.js
*   **Database:** PostgreSQL
*   **ORM:** Prisma
*   **Security:** JWT (JSON Web Tokens), bcrypt
*   **Deployment:** Railway / Render
